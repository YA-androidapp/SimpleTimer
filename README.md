SimpleTimer
====

## Overview
---

Androidデフォルトの時計アプリのタイマーが有効にされていない端末のための読み上げ機能つきタイマーアプリ

## Install
---

Download the APK package from [Google Play](https://play.google.com/store/apps/details?id=jp.gr.java_conf.ya.simpletimer)
